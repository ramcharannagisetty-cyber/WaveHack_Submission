Image Preprocessing
Raw images undergo white balance correction, lens distortion removal, and circular dish detection via Hough circle transform. CLAHE enhancement reveals low-contrast colonies invisible to the naked eye.
OpenCV · NumPy
🧠
Colony Segmentation
Background subtraction via large Gaussian blur separates colonies from agar. Adaptive Otsu thresholding + morphological cleanup + contour extraction provides robust colony boundaries across diverse media types.
cv2.findContours · Watershed
📐
Morphology Analysis
Each colony is characterized by 14 features: area, circularity, mean color (RGB), color variance, radius, eccentricity, and more. These feed a Random Forest that classifies target vs contaminant.
scikit-learn · scipy
